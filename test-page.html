🔧 Teste Técnico

Seu servidor está rodando em: http://localhost:3000

Endpoints disponíveis:

  • GET /health - Status do servidor
  • GET /widget.js - Widget JavaScript
  • GET /widget.css - Widget CSS
  • POST /api/describe-image - Descrever imagem
  • POST /api/summarize - Simplificar texto

🐛 Console de Debug

Pressione F12 para abrir o Developer Tools e veja os logs no Console.

📌 Dica: Se o widget não aparecer, verifique:

  • Se o servidor está rodando em http://localhost:3000
  • O console do navegador para mensagens de erro
  • Se a chave API do Google Gemini está configurada
