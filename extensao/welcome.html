🚀 Como Usar

  1. Navegar para qualquer página web
  2. O widget aparece automaticamente no canto inferior direito
  3. Clique no ícone para abrir o painel
  4. Use os recursos de acessibilidade que precisar

💡 Dica: Você pode clicar no ícone da extensão no navegador (no canto superior direito) para ativar/desativar o widget a qualquer momento.
